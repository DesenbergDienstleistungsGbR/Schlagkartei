# Mitteleinsatz Website (2 Seiten)

## Start lokal
Im Ordner ausführen:

python -m http.server 8000

Dann im Browser öffnen:
http://localhost:8000

## Nutzung
1) Auf der Startseite Erntejahr/Firma/Frucht wählen
2) Passende Schläge werden gelistet
3) Klick auf einen Schlag öffnet detail.html und zeigt die gefilterten Datensätze
